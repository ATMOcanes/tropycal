wind_to_category¶

tropycal.utils.wind_to_category(wind_speed)[source]¶

Convert sustained wind speed in knots to Saffir-Simpson Hurricane Wind Scale category.

Parameters

wind_speed (int) – Sustained wind speed in knots.

Returns

int – Category corresponding to the sustained wind. 0 is tropical storm, -1 is tropical depression.
